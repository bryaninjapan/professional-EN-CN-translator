# 测试指南

## 功能测试清单

### 1. 基础翻译功能
- [ ] 输入英文文本，选择中文，点击翻译
- [ ] 输入英文文本，选择日文，点击翻译
- [ ] 输入英文文本，选择韩文，点击翻译
- [ ] 验证翻译结果包含三个部分：翻译、术语表、难点解析
- [ ] 验证下载 Markdown 功能正常

### 2. 使用次数系统
- [ ] 首次访问，自动获得 3 次免费使用次数
- [ ] 使用翻译功能，次数正确递减
- [ ] 次数为 0 时，提示需要激活码或邀请码
- [ ] 刷新页面后，次数显示正确

### 3. 激活码系统
- [ ] 输入有效的激活码，成功激活并获得次数
- [ ] 输入无效的激活码，显示错误提示
- [ ] 重复激活同一激活码，显示已激活提示
- [ ] 激活码次数用完后，需要新的激活码

### 4. 邀请码系统
- [ ] 点击"邀请朋友"按钮，生成邀请码
- [ ] 使用他人的邀请码，双方各获得 3 次
- [ ] 尝试使用自己的邀请码，显示错误提示
- [ ] 重复使用同一邀请码，显示已使用提示

### 5. 错误处理
- [ ] 翻译失败时，使用次数自动恢复
- [ ] 网络错误时，显示友好提示
- [ ] API 错误时，显示详细错误信息

### 6. 后台管理
- [ ] 使用正确密码登录
- [ ] 使用错误密码登录，显示错误提示
- [ ] 创建激活码（单个和批量）
- [ ] 查看激活码列表和统计
- [ ] 删除激活码
- [ ] 查看使用统计
- [ ] 登出功能正常

## 边界情况测试

### 1. 空输入
- [ ] 空文本不能翻译
- [ ] 空激活码不能激活
- [ ] 空邀请码不能使用

### 2. 并发操作
- [ ] 快速连续点击翻译按钮
- [ ] 同时打开多个标签页

### 3. 数据一致性
- [ ] 使用次数计算准确
- [ ] 激活码状态正确
- [ ] 邀请码使用记录正确

- [ ] 页面加载时间 < 3 秒
