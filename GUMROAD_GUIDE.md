# Gumroad 账户和产品创建完整指南

## 第一步：创建 Gumroad 账户

### 1. 注册账户
1. 访问 [https://gumroad.com](https://gumroad.com)
2. 点击右上角 "Sign up" 或 "Get started"
3. 选择注册方式：
   - 使用邮箱注册
   - 使用 Google 账户
   - 使用 Facebook 账户
4. 填写必要信息并验证邮箱

### 2. 完善账户信息
1. 登录后，进入账户设置
2. 填写个人信息：
   - 姓名
   - 支付信息（用于收款）
   - 税务信息（如需要）

## 第二步：创建产品

### 1. 创建新产品
1. 登录 Gumroad 后，点击右上角头像
2. 选择 "Products" → "New product"
3. 选择产品类型：**Digital product**（数字产品）

### 2. 填写产品信息

#### 基本信息
- **产品名称**：`EN Translator License (100 uses)`
- **产品描述**：
  ```
  EN Translator 激活码
  
  包含：
  - 100 次专业翻译使用次数
  - 支持多语言翻译（中文、日文、韩文）
  - 三段式深度翻译（原文翻译、专业术语表、难点解析）
  
  购买后，激活码将通过邮件自动发送。
  ```

#### 定价设置
- **价格**：设置您的价格（例如：$9.99）
- **货币**：选择货币（USD/CNY/EUR 等）
- **折扣**：可选，设置促销折扣

#### 交付设置
- **交付方式**：选择 "Email"（邮件交付）
- **交付内容**：
  - 可以上传一个文本文件，包含激活码说明
  - 或者设置为 "No file"（无文件），手动发送激活码

#### 其他设置
- **产品图片**：上传产品封面图（可选）
- **产品链接**：Gumroad 会自动生成产品链接
- **SEO 设置**：填写产品关键词（可选）

### 3. 发布产品
1. 检查所有信息无误
2. 点击 "Save" 或 "Publish"
3. 产品发布后，会生成产品链接

## 第三步：获取产品链接

### 方法 1：从产品页面获取
1. 进入 "Products" 页面
2. 找到您的产品
3. 点击产品名称进入产品详情页
4. 复制浏览器地址栏的 URL

### 方法 2：从产品设置获取
1. 进入产品编辑页面
2. 在 "Share" 或 "Link" 部分可以看到产品链接
3. 链接格式通常为：
   - `https://your-username.gumroad.com/l/product-id`
   - 或 `https://gum.co/product-id`（如果设置了短链接）

## 第四步：配置到代码中

### 更新购买链接
1. 打开 `app/page.tsx` 文件
2. 找到购买模态框（约第 754 行）
3. 将 `href` 属性中的链接替换为您的 Gumroad 产品链接：

```tsx
<a
  href="https://your-username.gumroad.com/l/your-product-id"  // ← 替换这里
  target="_blank"
  rel="noopener noreferrer"
  className="..."
>
  前往 Gumroad 购买
</a>
```

### 示例
如果您的 Gumroad 产品链接是：
```
https://johndoe.gumroad.com/l/en-translator-license
```

则代码应该是：
```tsx
<a
  href="https://johndoe.gumroad.com/l/en-translator-license"
  target="_blank"
  rel="noopener noreferrer"
  className="..."
>
  前往 Gumroad 购买
</a>
```

## 第五步：设置自动交付（可选）

### 方案 1：手动发送激活码（推荐）
- 产品设置为 "No file"（无文件）
- 购买后，Gumroad 会发送确认邮件
- 您在后台手动创建订单并发送激活码

### 方案 2：自动发送文本文件
1. 创建一个文本文件（.txt），内容为：
   ```
   感谢您的购买！
   
   您的激活码将在 24 小时内通过邮件发送。
   如有问题，请联系我们。
   ```
2. 上传为产品文件
3. 购买后自动发送

## 工作流程

### 完整流程
1. **用户购买**
   - 用户点击"购买激活码"
   - 跳转到 Gumroad 产品页面
   - 完成支付

2. **收到通知**
   - Gumroad 发送销售通知邮件给您
   - 邮件包含：订单号、客户邮箱、金额等信息

3. **管理员处理**
   - 登录后台管理系统 (`/admin`)
   - 进入"Gumroad 订单"标签页
   - 点击"创建订单"
   - 填写订单信息（从 Gumroad 邮件中获取）
   - 点击"生成激活码"
   - 复制激活码并发送给客户

4. **用户激活**
   - 用户收到激活码邮件
   - 在主页面输入激活码
   - 激活成功，获得 100 次使用次数

## 收款设置

### 设置收款账户
1. 进入 Gumroad 账户设置
2. 选择 "Payouts"（收款）
3. 添加收款方式：
   - PayPal
   - 银行账户
   - Stripe（如果可用）
4. 验证收款账户

### 收款周期
- Gumroad 通常每周或每月结算一次
- 首次收款可能需要验证账户

## 常见问题

### Q: 如何查看销售记录？
A: 登录 Gumroad → "Sales" 页面，可以看到所有销售记录。

### Q: 如何获取订单号？
A: 在 Gumroad 的销售通知邮件中，或在 "Sales" 页面查看订单详情。

### Q: 可以设置自动发送激活码吗？
A: 可以，但需要集成 Gumroad API 或使用 Zapier 等自动化工具。当前版本建议手动发送。

### Q: 如何修改产品价格？
A: 进入产品编辑页面 → "Pricing" → 修改价格 → 保存。

### Q: 如何设置折扣码？
A: 进入产品编辑页面 → "Discounts" → 创建折扣码。

## 安全建议

1. **保护账户安全**
   - 使用强密码
   - 启用双因素认证（2FA）
   - 定期检查账户活动

2. **验证收款账户**
   - 确保收款账户信息正确
   - 及时验证新添加的收款方式

3. **订单验证**
   - 收到订单后，验证支付状态
   - 确认客户邮箱正确
   - 再发送激活码
